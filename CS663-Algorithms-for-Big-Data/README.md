# Problem Set 1

## Setup

Create a virtual environment and install dependencies:

```bash
cd CS663-Algorithms-for-Big-Data
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Running the Code

Activate the virtual environment (if not already active):

```bash
cd CS663-Algorithms-for-Big-Data
source .venv/bin/activate
```

Run each problem:

```bash
cd CS663-Algorithms-for-Big-Data
cd ProblemSet1/problem_1 && python problem_1.py && cd ../..
cd ProblemSet1/problem_3 && python problem_3.py && cd ../..
cd ProblemSet1/problem_4 && python problem_4.py && cd ../..
```

Note: problem 2 has no code, just a markdown of the proof.
